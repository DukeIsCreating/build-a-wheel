Instructions

  1. Create cutting project in the software for your cutting machine. Take advantage of the fact that two dampeners can be cut from a single 9"x12" sheet of felt.
  2. Attach felt to a strong grip cutting mat, and tape down using 3/4" masking tape. The adhesive backing for the felt does not like to adhere to the cutting mat.
  3. Adjust cutting settings so that both the felt and the adhesive backing are cut completely through, so the adhesive-backed dampeners can be positioned freely in a later step.
